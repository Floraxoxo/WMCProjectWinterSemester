Selectors

As already mentioned in the introduction there are many different selectors. But what exactly is a selector? A selector is a pattern used to select the elements of a document you want to style. Basically they are used to target HTML elements on a web page.
However, there are many different types of selectors:

Basic Selectors

If you want to select...

* {
color: red;
}
Selects all items. In this case it colors every element red
div * {
color: red;
}
Selects all items inside a certain element. In this case it colors every p element inside a div red.

p {
color: red;
}
Selects all p elements. In this case it colors every p element red.

Classes & IDs

* {
color: red;
}
Selects all items. In this case it colors every element red
div * {
color: red;
}
Selects all items inside a certain element. In this case it colors every p element inside a div red.

CSS:
#nameOfAnID {
color: red;
}
HTML:
< p id="nameOfAnID" >hello < /p >
Selects all p elements. In this case it colors every p element red.
